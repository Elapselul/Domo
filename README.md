# DOMO

Custom Raspberry Pi vehicle monitoring dashboard for a 2018 Isuzu D-Max.

## Current features

- PyQt6 dashboard
- Simulated vehicle data
- Custom boost gauge
- Stealth theme
- Modular app structure

## Run the app

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python main.py
